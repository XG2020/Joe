## Joe 魔改 2.0
## 🌟 主要特性
- 修复了一些bug
- 优化代码逻辑
- 增添了许多新功能

### 🎨 视觉设计
- **现代化界面** - 采用扁平化设计风格，支持明暗模式切换
- **响应式布局** - 适配桌面、平板、手机等各种设备
- **丰富动画效果** - 内置多种炫酷的列表动画和过渡效果
- **自定义主题色** - 支持自定义网站主色调和配色方案

### 🚀 性能优化
- **轻量级代码** - 优化的CSS和JavaScript，加载速度极快
- **图片懒加载** - 支持图片懒加载，提升页面加载性能
- **CDN加速** - 支持静态资源CDN加速
- **代码压缩** - 自动压缩CSS和JS文件

### 📱 交互功能
- **轮播图模块** - 支持首页轮播图展示
- **热门文章** - 自动展示热门文章列表
- **推荐文章** - 支持手动设置推荐文章
- **评论系统** - 集成强大的评论功能
- **点赞功能** - 文章点赞和统计功能

### 🎯 个性化定制
- **多种布局** - 支持多种首页布局风格
- **自定义导航** - 灵活的导航栏配置
- **侧边栏小工具** - 丰富的侧边栏组件
- **背景特效** - 6种背景动画效果可选
- **鼠标特效** - 11种鼠标光标样式

## 📦 安装方法

1. **下载主题**

2. **上传到Typecho**
   - 将主题文件夹重命名为 `Joe`
   - 上传到Typecho的 `/usr/themes/` 目录

3. **启用主题**
   - 登录Typecho后台
   - 进入「控制台」→「外观」
   - 找到Joe主题并点击启用
